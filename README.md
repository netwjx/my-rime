rime 鼠须管输入法配置

使用方式

    git init
    git pull https://github.com/netwjx/my-rime.git master

主要特点

* 双拼 小鹤方案
* F4唤起
* 半角符号(英文符号)
* shift切换英文和上屏
* 每页9个候选
* mojave 暗夜皮肤
  * 横向展示
  * 非嵌入式编码行
